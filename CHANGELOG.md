# Changelog

All notable changes to THE ARENA Mobile app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-18

### Added
- Initial release of THE ARENA Mobile
- Quad-exposure to 4 AI models (Gemini, Grok, Claude, ChatGPT)
- Multi-channel publishing (Twitter/X, Substack, Email)
- Blockchain seniority claiming (Solana, Base)
- Offline-first architecture with cached responses
- History management with delete and share functionality
- Settings screen with API URL configuration
- Deep linking support (thearena://)
- Pull-to-refresh on history screen
- Copy responses to clipboard
- Haptic feedback throughout the app
- Loading animations and states
- Error boundary for graceful error handling
- Accessibility labels for screen readers
- GitHub Actions workflow for automated APK builds
- One-touch deploy script for Android

### Technical
- React Native + Expo SDK 50
- TypeScript for type safety
- Zustand for state management
- AsyncStorage for local persistence
- expo-router for navigation
